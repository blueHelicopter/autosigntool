## autosigntool (EN)
Tool for automating processing of occupational safety assessment documents.

Link to the latest version: https://github.com/blueHelicopter/autosigntool/releases/latest

### Features
- Converts .doc to .docx
- Splits documents into individual cards
- Automatically inserts signatures from PNG files
- Sets signing dates
- Exports final documents to PDF

### Requirements
- Windows
- Microsoft Word installed

### Usage
Download the .exe file and run it.  
Follow the instructions in the console.

Full instructions are available in the attached README.txt

## autosigntool (RU)
Автоматическая вставка подписей и дат в карты специальной оценки условий труда.

Ссылка на последнюю версию программы: https://github.com/blueHelicopter/autosigntool/releases/latest

### Возможности
- Конвертация .doc в .docx
- Разделение документов на индивидуальные карты
- Автоматическая вставка подписей из PNG-файлов
- Вставка даты подписания
- Экспорт готовых документов в PDF

### Требования
- Windows
- Установленный Microsoft Word

### Использование
1. Скачайте файл `autosigntool.zip` из раздела [Releases](https://github.com/blueHelicopter/autosigntool/releases/latest)
2. Распакуйте архив
3. Запустите `autosigntool.exe`
4. Следуйте инструкциям в консоли
